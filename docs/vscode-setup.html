2. NAS-Verbindung einrichten (Remote SSH)

1 Ctrl+Shift+P > tippe: Remote-SSH: Add New SSH Host
2 Eingeben: ssh dole4711@192.168.178.28
3 Config-Datei waehlen: C:\Users\ralph\.ssh\config
4 Ctrl+Shift+P > Remote-SSH: Connect to Host > 192.168.178.28
5 Ordner oeffnen: /volume1/docker/cloudbot/
SSH-Key ist bereits eingerichtet - keine Passwort-Eingabe noetig!

3. Lokales Projekt oeffnen

1 File > Open Folder > C:\Users\ralph\claude\Neugruendung
2 Terminal oeffnen: Ctrl+Oe (oder Ctrl+`)
